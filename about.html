About
This Spotify Music Tracker web page is a project that returns information about a user. The information displayed varies from the user's most listened song or album, some of their favorite artists, their profile link and more. The project utilizes Spotify's API in order to access and return the information displayed here.

Guide
This web page uses Vite to utilize the API, so start by installing vite. After, run 'npm run dev', follow the link provided and access the website.
